Visual Memory Game

Hello! We appreciate you playing this game to help us better understand what makes some images more memorable than others.

Instructions

Images will be displayed one-by-one, for a second each. Some images will be displayed twice. Press the space bar if you've seen the image before.

The game is split up into levels. Play as many as you would like! You may quit at any time. Thanks for playing!
